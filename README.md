# dbms_finalproject

## 題目: 健身房會員管理系統
   Database: **SQLite** <br>
   Back-End Framework: **flask**

### *Pre-Start*
  至 https://sqlite.org/download.html 下載SQLite

### *Start*

#### *Back-End*

```shell
cd backend
```
```shell
pip install -r requirements.txt
```
```shell
python db.py
```
```shell
python app.py
```
```shell
curl "http://localhost:5000"
```
若 *StatusCode* 為 *200*，且 *Content* 為 *Hello World* 則表示連接成功
